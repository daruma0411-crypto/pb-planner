### 4-2. 空白地帯 3 パターン

#### 空白①: 「Woman-friendly UX 標準装備 100L」（T1 ×K1×K6×K3）

- **空白の根拠**: 競合右上ゾーン（高ユーザビリティ×大容量）で UX 訴求は「液晶」「冷却ファン」止まり（3C 2-1）。蓋アシスト・排気ディフューザー・大型文字UI・SOP同梱を**標準装備で束ねた 100L 機は競合に未確認**。
- **自社が埋める手段**: トミー精工 OEM 柔軟性＋プロシェア／バスケット／教育コンテンツのバンドル。
- **リスク**: 量産実績これから（K1 充足度○）、価格上昇とヤマト HVA-110LB ¥742,000 との乖離。

#### 空白②: 「省フットプリント 100L 縦型 × 設置事前検証」（T2 ×K2×K3）

- **空白の根拠**: 「100L 縦型が欲しいが設置スペース足りない」が SNS / P3 で頻出（3C 1-5）。競合各社のフットプリント最適化訴求はデータに記載なし。100L 引き合いは 1.4倍 で需要は明確。
- **自社が埋める手段**: AXEL 連携の設置シミュレーター＋カタログ配荷の設置事例ナレッジ蓄積（KSF K2 は構想段階）。
- **リスク**: 設計はOEM依存、シミュレーター実装は未確認。

#### 空白③: 「Woman-friendly UI × バリデーション証跡」両立 100L（T3 ×K1×ロギング）

- **空白の根拠**: 競合のバリデーション証跡対応は限定的（ペイン#5・中強度）、かつ woman-friendly UX 軸は競合各社未着手（3C 2-6）。**この2軸を同時に満たす 100L 機は競合・自社既存ともデータ上に存在せず**。
- **自社が埋める手段**: トミー BSX-500 系の 7インチタッチパネルをベースに、大型文字＋ハイコントラスト＋ロギング出力をPB仕様で指定（実装可否はデータに記載なし → 二次調査要）。
- **リスク**: 製薬の仕様審査ハードル高、開発コスト・LT増大。
