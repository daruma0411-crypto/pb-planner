## 5. データ制約に関する明記

- `asone_part_no` `jan_code` `maker_part_no`（PB専用型番） `cost` `gross_margin_rate` `spec_diff_capacity`（有効容量L表記）はいずれも前段レポートに記載なし。商品マスタ JSON / CSV 上は「データに記載なし」と明記し、アクションプラン M-D07 / M-P02 / M-P07 で確定するフローとした。
- `price_list_excl_tax_jpy` はレンジ表記（¥880,000〜¥950,000）。原価積み上げ二次調査（M-P02）完了後に単一値で確定要。
- 蓋アシスト機構・排気ディフューザーの具体方式は 4P 1-3 R1/R2 で「OEM 協議」「安全性二次調査要」と明記されているため、本レポートではコピー・マスタ上「機構あり／標準装備」までの粒度に留めた。
- キャッチコピー CC1〜CC5 は前段の訴求軸 A1〜A5（4P 4-1）と STP T1 ポジショニング・3C VOC（1-5）の範囲内で構成し、新規ファクト（具体的な軽量化数値、騒音dB値、温度域拡張等）は一切付加していない。
