3C 分析レポート: demo-full-v2

対象カテゴリ: 高圧蒸気滅菌器（オートクレーブ）／PB コンセプト: woman-friendly 100L／ベース機種候補: トミー精工 FLS-1000

1. Customer（顧客分析）

1-1. 市場規模・成長性

国内研究用オートクレーブ市場は、大学・公的研究機関、製薬・バイオ企業、食品検査、医療施設（クリニック・歯科を除く研究室用途）の 4 大セグメントを中心に、年間台数ベースで概ね横ばい〜微増の成熟市場である一方、100L クラス（縦型 / 大容量帯） は、再生医療・細胞培養・ワクチン研究の拡大、サステナビリティ要請（培地廃棄物の一括滅菌ニーズ）を背景に伸長傾向にある。アズワン POS の手入力サマリでも、100L クラスの引き合いは直近 12 ヶ月で 1.4 倍 に拡大しており、PB 投入の市場タイミングとして合理性が高い。具体的な金額ベースの市場規模はデータに記載なし。

アズワン取扱の研究用滅菌器カテゴリ全体では、月間平均 35 台・価格帯 ¥40-90 万が約 60% を占めるボリュームゾーンを形成。リピート率は大学 30%・製薬 50% と、製薬セグメントで指名買いが起こりやすい構造である。

1-3. JTBD（Job To Be Done）

「安全に、一人でも、想定どおりに、繰り返し、培地・器具・廃棄物を滅菌完了状態で取り出したい」

機能ジョブ＝確実な滅菌、感情ジョブ＝「蒸気・重さへの恐怖がなく安心して回せる」、社会ジョブ＝「ラボ DX・女性活躍・サステナの実践として上司・施設に説明できる」。

2. Competitor（競合分析）

2-1. 競合マッピング図記述

縦軸＝「ユーザビリティ／女性配慮（軽い開閉・UI・安全機能）」、横軸＝「容量・拡張性（〜110L＋プログラム数）」で配置した場合：

  • 右上（高ユーザビリティ × 大容量）: ヒラヤマ HV-110ⅡLB、ヤマト HVA-110LB、トミー LPS-700／FLS-1000 が密集する 激戦ゾーン。
  • 右下（機能リッチだが重操作）: アルプ CLG-DVP40L（117L）、MCS-40L（118L）。SUS316L 等の素材優位、サニタリ志向だが UI 訴求は限定的。
  • 左上（小型 × 使いやすさ）: ヒラヤマ HV-25Ⅱ、HF220、ヤマト ST201。歯科・小規模ラボ向け。
  • 左下（簡易圧力容器）: アルプ KTR/KYR 系、卓上低価格帯。

PB が狙う 「woman-friendly × 100L」 は、右上の中でも UX 軸で空白の象限であり、現状は単機能の「液晶パネル」「冷却ファン標準装備」程度の差別化に留まる。

2-3. 各社訴求メッセージ（公開仕様から推定）

  • トミー精工: 「タッチパネル UI ＋ 4〜5 コース」「冷却ファン標準」── プログラム多様性／ラボ DX 訴求。
  • ヒラヤマ: 「5 コース 8 メモリー」「フタロック可変式」── 培地・寒天・液体・廃棄物などプロセス専門性訴求。
  • ヤマト科学: 「価格／容量比（HVA-110LB ¥742,000）」── コストパフォーマンス訴求。100L 級では最安帯。
  • アルプ: 「SUS316L 缶体」「前面扉式（横開き）」「100〜140℃ 広温度域」── サニタリ／高温域・横開き UX 訴求（女性配慮に最も近いが UI 訴求は弱い）。

2-4. シェア推定

公式シェア数値はデータに記載なし。アズワン取扱現場感（POS サマリ＋カタログ露出）からの定性推定は、研究用 50〜110L 帯で ヒラヤマ ≈ ヤマト > トミー > アルプ > その他。100L 級単体では ヒラヤマ HV-110Ⅱ／ヤマト HVA-110LB が定番化しているとみられる。

2-5. サポート密度

  • ヒラヤマ・ヤマトは販社経由のメンテ網が広く、アズワン経由販売との親和性高。
  • アルプは横開き機構や SUS316L 等のサニタリ仕様で製薬・食品 GMP ラインに食い込む一方、消耗品供給の現場声はデータに記載なし。
  • トミー精工はモデル数が豊富（簡易圧力容器〜レトルト殺菌 SR-240 まで）、製造パートナーとしての OEM 柔軟性が高い と推定。

2-6. 直近 12-24 ヶ月動向

  • 100L 級引き合いの 1.4 倍（アズワン POS サマリ）── 再生医療・培地廃棄物一括処理需要を反映。
  • タッチパネル UI 標準化（トミー LPS シリーズ、BSX-500）── UI 競争が次フェーズに。

3. Company（アズワン PB ブランド = AS ONE + ナビス）

3-1. PB ブランドの強み

  1. 流通カバレッジ: アズワン本体のカタログ（紙＋AXEL）＋ナビスの医療チャネルにより、大学・公的研究・製薬・食品・医療研究室まで 5 セグメント横断の配荷力。POS で平均月 35 台・¥40-90 万帯 60% の販売実績はオートクレーブ周辺のクロスセル基盤になる。
  2. 周辺消耗品の自社ラインアップ: 滅菌バッグ「プロシェア SPS-1〜SPS-7」（7 サイズ）、超音波洗浄器 MCS シリーズ（2〜27L）、ガラス容器洗浄機 GS-01、フラスコブラシ、プラ角型バスケット（浅／深型 ×小/中/大）など、前処理〜滅菌〜後工程を 1 社で揃えられる稀有なポジション。「100L PB 本体 ＋ プロシェア滅菌バッグ ＋ バスケット」のバンドル提案が可能。
  3. PB 価格力: アズワン PB は同等仕様の競合品より概ね 10〜20% の値ごろ感を訴求しやすく、本案件のターゲット価格帯 ¥70〜100 万（ベース機種 FLS-1000 ¥1,080,000 を基準に最適化）でも勝負できる。

3-2. 既存ラインアップとの共食い検証

  • アズワン PB の 自社ブランド大型オートクレーブは現提供データ上に存在せず、滅菌器本体は他社品（トミー、ヒラヤマ、ヤマト、アルプ）の流通取扱中心と推定。よって 100L 級 PB は社内カニバリゼーションのリスクが低い。
  • ただし AXEL 上での ヤマト HVA-110LB（¥742,000）／ヒラヤマ HV-110ⅡLB（¥965,000） との価格・スペック競合は不可避。「woman-friendly 100L」という UX 差別化軸で別ジョブとして定位させることが共食い回避の鍵。

3-3. 販社チャネル適合性

  • 大学・国研：AXEL／代理店経由でカタログ載録による即引合化が容易。
  • 製薬・バイオ：リピート 50% のセグメントで「PB ＋ 周辺消耗品定期購買」モデルが効く。
  • 食品検査・医療研究室：ナビス導線で看護研究室・SMO 系まで展開余地。

3-4. 製造パートナー（補足）

ベース機種候補は トミー精工 FLS-1000（¥1,080,000、缶体Φ450×678mm、105〜123℃、0.193MPa、電気ヒーター 3.5kW、小型圧力容器）。同社は LPS-300/500/700、MPS-510/710、BSX-500、ES/LBS 系、レトルト SR-240 まで広い製品系列を持ち、OEM/仕様カスタム（UI 文字サイズ、蓋アシスト機構、排気ディフューザーなど）の受容性が高い と推定（製造パートナーとしての適合）。
